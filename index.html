Exercice pour indenter correctement son code

1) Crée un dossier "exo-indentation" et crée un fichier index.html dans le dossier créé.

2) Glisse le dossier dans Atom.

3) Télécharge le fichier code.txt

4) Ouvre et copie colle le contenu du fichier code.txt dans ton fichier index.html

5) Présente correctement le code en ajoutant la bonne indentation avec des tabulations.
